Modern DevOps with GCP
4:10 PM

In this talk I will discuss different ways of developing and operating infrastructures for web services, also referred to as DevOps. This will include topics such as infrastructure provisioning, continuous integration, application deployment, and monitoring. GCP services such as load-balancing and BigQuery, which are useful for improving efficiency of various DevOps processes, will also be covered. Finally, I will introduce Mackerel, a server monitoring SaaS that functions as a hub of actions pertaining to DevOps.

ทางเลือกใหม่ของ DevOps บน GCP
EN
16:10 น.

ในการบรรยายนี้เราจะพูดถึงทางเลือกต่างๆในการพัฒนา และจัดการโครงสร้างระบบสำหรับ Web Service ในมุมมองของ DevOps หัวข้อนี้จะรวมไปถึง infrastructure provisioning, continuous integration, application deployment และการ monitor บริการบน GCP เชjน load-balancing และ BigQuery ซึ่งจะช่วยเพิ่มประสิทธิภาพการทำงานให้กับ DevOp และบริการใหม่ Mackerel ที่จะช่วยในการ monitor ระบบต่างๆอีกด้วย

Your Data and the World Beyond MapReduce
4:55 PM

Google's MapReduce has been the most important innovation in the Big Data industry in last decade, which provided the technical foundation for Apache Hadoop MapReduce and other solutions. But at Google, we have moved to next generation technologies such as Dremel, Flume and MillWheel that replace the classic MapReduce infrastructure. In this session, we will introduce two products from Google Cloud Platform, BigQuery and Cloud Dataflow, that allow developers to leverage the power of the Google's latest Big Data technologies.

ข้อมูลของคุณ กับโลกที่ก้าวไกลไปกว่า MapReduce
EN
16:55 น.

เทคโนโลยี MapReduce จาก Google เป็นส่วนสำคัญในนวัตกรรม Big Data ของอุตสหกรรมตลอด 10ปีที่ผ่านมา ซึ่งทำให้เกิด Apache Hadoop MapReduce และทางเลือกอื่นๆอีกมากมาย ที่ Google เราก้าวไปไกลกว่าเดิม ด้วยเทคโยโลยีใหม่เช่น Dremel, Flume และ MillWheel ที่เข้ามาแทนที่โครงสร้าง MapReduce แบบเดิมๆ ในหัวข้อนี้เราจะนำเสนอ 2 ผลิตภัณฑ์จาก Google Cloud Platform นั้นก็คือ BigQuery และ Cloud Dataflow ที่จะช่วยนักพัฒนาในการยกระดับขุมพลัง Big Data ล่าสุดจาก Google

คุณ Sato เป็น Developer Advocate ที่ทีม Google Cloud Platform ดูแลและสนับสนุนนักพัฒนา ชุมชนนักพัฒนา Google Cloud Platform ในทวีปเอเซีย เช่น GCP User Group (GCPUG), #bq_sushi และ gcp ja night และยังสนับสนุนการเปิดตัวผลิตภัณฑ์ Big Data เช่น Cloud Dataflow และ BigQuery อีกด้วย

Shinji Tanaka
Google Developer Expert

Shinji Tanaka is the Chief Technology Officer at Hatena Co., Ltd. Hatena provides a variety of web-based services including Hatena Blog, and Hatena Bookmark. Launched in 2014, Hatena’s newest service, Mackerel, is a cloud management SaaS. Shinji is responsible for the technology behind all of Hatena’s services from front-end to infrastructure.

คุณ Tanaka ดำรงตำแหนแหน่ง CTO ให้กับบริษัท Hatena เป็นผู้ให้บริการเว็บไซต์ที่หลายหลาย เช่น Hatena Blog และ Hatena Bookmark และได้เปิดตัวบริการใหม่ Mackerel เป็นบริการจัดการระบบ Cloud SaaS ในปี 2557 เขาเป็นผู้ดูแลเทคโนโลยีที่อยู่เบื้องหลังทั้งหมดของบริการของ Hatena ตั้งแต่ระบบหน้าบ้านจนไปถึงโครงสร้างระบบทั้งหมดเลยทีเดียว
